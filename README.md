# game-mate-backend

**Features**:

- Spring Boot 3
- Kotlin
- Gradle
- OAuth2

## Setup

- Clone this repo

```shell
git clone https://github.com/Perfect-Paradise/game-mate-backend
```

- JDK version: 17 or 21 (my personal choice: openjdk-21.0.2)

## Run

### Commend Line

```shell
./gradlew bootRun
```
